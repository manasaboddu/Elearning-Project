Introduction:

Javascript is a client side , object based scripting language.Javascript is an Interpreted language.
Scripting language is any type of language, including JavaScript, that is capable of programmatically controlling a web page or returning some sort of response to a web browser.

JavaScript Advantages:

  1. JavaScript is used for Client-side validation
  2. JavaScript is used to enhance the functionality and appearance of web pages.
  3. JavaScript makes the web pages more interactive and dynamic and it provides the programming foundation for the server-side scripting
