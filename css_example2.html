This page explains the CSS background properties.

The background CSS property is used for setting the individual background values in a single place in the style sheet. There are 8 different background properties that affects the HTML elements.

  1. Background-clip:This CSS property lets you control how far a background image or color extends beyond an element's padding or content.
  2. Background-color:This CSS property sets the background color of an element, either through a color value or the keyword transparent.
  3. Background-image:This CSS property applies a graphic (e.g. PNG, SVG, JPG, GIF, WEBP) or gradient to the background of an element.
  4. Background-origin:This CSS property determines the background positioning area, that is the position of the origin of an image specified using the background-image CSS property.
  5. Background-position:This CSS property sets the initial position for each defined background image, relative to the background position layer defined by background-origin.
  6. Background-repeat:This CSS property defines how background images are repeated. A background image can be repeated along the horizontal axis, the vertical axis, both axes, or not repeated at all.
  7. Background-size:This CSS property specifies the size of the background images. The size of the image can be fully constrained or only partially in order to preserve its intrinsic ratio.
  8. Background-attachment:This CSS property determines whether that image's position is fixed within the viewport, or scrolls along with its containing block.
