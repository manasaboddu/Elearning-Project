What is HTML?!

HTML-Hyper Text Markup Language.It describes the structure of a webpage.The HTML elements are reprersented by its "tags".

  • All HTML elements can have attributes.
  • Attributes provide additional information about an element.
  • Attributes are always specified in the start tag.
  • Attributes usually come in name/value pairs like: name="value"
  • Setting the style of an HTML element, can be done with the style attribute.
  • The background-color propertydefines the background color for an HTML element.
  • HTML also defines special elementsfor defining text with a special meaning.
  • HTML links are hyperlinks.
  • The href attribute specifies the destination address of the link.
  • An HTML table is defined with the <table> tag.
  • Each table row is defined with the <tr> tag. A table header is defined with the <th> tag. By default,table headings are bold and centered. A table data/cell is defined with the <td> tag.
