Lets Play With Strings.

This is similar to missing letters game.
Here we have used javascript functionalities to determine if the missing letters entered by the user matches with the correct letters.
If the letters doesnt match with the correct letters, it checks and display the error message.









In the above example, we have used the text boxes to display the letters.
The missing letters are stored in an Array for each word.
When the user enters the missing letters and click on check button,the eventlisteners and and
'getElementId' properties takes the user input and store it in variables and then compares with the letters stored in an array.
It checks if the missing letters entered matches with correct letters or not.
